---
description: Start the ADK LLM Proxy server with OpenAI
---

Start the ADK LLM Proxy server on port 8000 with OpenAI as the provider and gpt-4o-mini as the model.

Use the Bash tool to run:
```bash
python main.py -provider openai -model gpt-4o-mini
```

After starting, the server will be available at http://localhost:8000
